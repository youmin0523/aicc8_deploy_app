# 기능 리스트 (Feature List)

## 1. 인증 (Authentication)

- **Google 로그인**: `@react-oauth/google`을 사용한 Google 계정 로그인 지원.
- **로그아웃**: 현재 로그인된 계정에서 로그아웃 기능.
- **사용자 정보 표시**: 로그인 시 사용자 이름 표시.
- **접근 제어**: 로그인하지 않은 경우, 할 일 목록 대신 "로그인이 필요한 서비스입니다" 메시지 표시.

## 2. 대시보드 및 네비게이션 (Dashboard & Navigation)

- **사이드바 네비게이션**: 주요 메뉴로 이동할 수 있는 사이드바 제공.
- **라우팅**:
  - **Home (`/`)**: 모든 할 일 목록 표시.
  - **Completed (`/completed`)**: 완료된 할 일 목록 표시.
  - **Important (`/important`)**: 중요 표시된 할 일 목록 표시.
  - **Proceeding (`/proceeding`)**: 진행 중인(미완료) 할 일 목록 표시.
- **반응형 레이아웃**: 다양한 화면 크기에 대응하는 레이아웃.

## 3. 할 일 관리 (Task Management)

- **할 일 조회 (Read)**:
  - 사용자별 할 일 목록 데이터베이스에서 조회 (`GET`).
  - 로딩 중 스켈레톤 UI (`LoadingSkeleton`) 표시.
  - 완료 여부 및 중요도에 따른 필터링 보기 제공.
- **할 일 추가 (Create)**:
  - "할 일 추가하기" 버튼을 통한 모달 창 활성화 (`Modal`).
  - 입력 필드: 제목, 날짜, 설명, 중요 여부, 완료 여부.
  - 데이터베이스에 새 할 일 저장 (`POST`).
- **할 일 수정 (Update)**:
  - 기존 할 일 내용 수정 (`PUT`).
  - **완료 상태 토글**: 할 일의 완료/미완료 상태 변경 (`PATCH`).
  - **중요 상태 토글**: 할 일의 중요/일반 상태 변경.
- **할 일 삭제 (Delete)**:
  - 특정 할 일 삭제 기능 (`DELETE`).

## 4. UI/UX

- **모달 (Modal)**: 할 일 추가 및 수정을 위한 팝업 인터페이스.
- **알림 (Toast)**: 작업 성공/실패 시 `react-toastify`를 이용한 알림 메시지 제공.
- **다크 모드 테마**: 어두운 배경과 대비되는 텍스트 색상을 사용한 현대적인 디자인 (`#212121` 등 사용).
